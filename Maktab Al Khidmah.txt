Data model:
Akun:
- AkunID (number)
- NamaAkun (varchar)
- NoHP (varchar)
- Peran (number)
- DaerahID (null selain PD, KR, J)
- RombonganID (null selain J)
- Aktivasi (always 1 selain PD, PDU, PM, PP)
- latitude
- longitude
- keterangan (untuk PDU, PM, PP / tugas spesifik)


Rombongan:
- RombonganID (number)
- NamaRombongan (varchar) generated
- Koor (AkunID)
- NamaKendaraan (varchar)
- JenisKendaraan (Bis Besar/Kecil, Mobil pribadi, Elf)
- NoKendaraan (varchar)
- JumlahJamaah (number)
- WaktuBerangkat (timestamp)
- WaktuTiba (timestamp)
- Kategori (VVIP, VIP, Santri, Umum)
- AlokasiMaktab (MaktabID)
- AlokasiParkir (ParkirID)

Maktab:
- MaktabID
- NamaPemilik
- NoHP
- Alamat
- Kategori (VVIP, VIP, Santri, Umum)
- Latitude
- Longitude
- Kapasitas
- Aktivasi (Aktif untuk dialokasikan, nonaktifkan)
- RiwayatAlokasiDaerah (DaerahID)

Parkir:
- ParkirID
- NamaTempat
- Alamat
- Latitude
- Longitude
- Status (Masih tersedia, Penuh)
- Kategori (VVIP, VIP, Santri & Umum)

PengambilanMakanan:
- RombonganID
- JadwalMakanID
- WaktuPengambilan
- Status (tersedia / belum diambil, sudah diambil)

JadwalMakan:
- JadwalMakanID
- JudulJadwal
- Waktu

Daerah:
DaerahID
Nama

Stack:
- Webserver : Flask
- Database : SQLite
- Libraries : 

Login : NoHP, Peran -> Akun anda sedang dipakai orang lain saat ini, hubungi admin / maaf no hp tidak ada / maaf peran tidak sesuai / berhasil
Daftar/Edit : Nama, NoHP, Peran, untuk PD & KR Pilih daerah / usul daerah baru (nulis) -> Ok, if edited reset activation
View: Nama, NoHP, Peran, Status (teraktivasi, belum teraktivasi (hubungo admin), ditolak (keluar & buat akun lg))
